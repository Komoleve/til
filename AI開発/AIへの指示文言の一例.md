# AIへの指示文言の一例

## アイデア出し
- ～したい。複数案（または具体的に○個）だして

## 要件定義
- 要件定義したい。不明点をつぶしたいから、わからないところどんどん質問して。

## 要件定義書を作るとき
- 要件定義書として保存したいので不要な文章は全部削除
  - 最初と最後の挨拶は不要
  - 純粋な要件定義書の文書として出力して

## 開発開始時
- この要件定義書の内容を基に開発を開始してください
  - pip install 必要なものは `/requirements.txt` にまとめてください

## README を作るとき
- `README.md` を作成してください
  - 利用者向けの内容を作成して
  - 開発者向けの内容は不要

## TIL を整理するとき
- `.txt` に各種技術メモを書いています
  - これを `.md` に変換してわかりやすいファイル名にして
  - さらにそれを日本語フォルダで整理して
